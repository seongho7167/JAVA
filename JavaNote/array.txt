## array 

(javascript)   An array is a special variable, which can hold more than one value 

하나 이상의 값을 붙잡고있을수 있는 특별한 변수를 배열이라 한다.


(JAVA)    Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

각각의 값을 분리된 변수로 선언하는 대신 한 변수에 여러 값을 저장하기 위해 사용되는 것이 배열이다.
