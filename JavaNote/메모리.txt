## static

메모리에 미리 할당하기 위해 사용하는 지정자

메모리 영역 : 3가지로 구분

Class 메모리: class 저장
클래스 사용 
    - 기본: 객체 생성후 사용
    - 멤버: Object.MV(객체.멤버변수), Object.MM(객체.멤버메서드)
    primitive 타입은 값을 저장
    reference 타입은 주소를 저장
    - 형식 : DataType 변수명 = 값

    클래스타입 변수명 = new 클래스타입();  = new 생성자;            // 클래스명과 동일한 이름을 메서드를 생성자라고 함


stack 메모리: 변수

hip 메모리
