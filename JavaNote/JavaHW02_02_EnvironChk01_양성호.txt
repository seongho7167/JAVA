##JDK 설치 - 환경설정

00. DOS 명령어: 
    - 외부 명령어: 사용빈도가 작고, 크기가 커서 메모리에 올려놓기 어려운 실행명령어 >> 해당 폴더에서 실행
    - 내부 명령어: 사용빈도가 높은 명령어, 명령어 크기가 메모리에 올라갈수 있을정도로 작은 크기 >> 현재 폴더 위치와 상관없이 실행 가능(cls, cd, dir...)

01. javac
   - C:\Java1.8\jdk1.8\bin
   - cd \
   - 'javac'은(는) 내부 또는 외부 명령, 실행할 수 있는 프로그램, 또는
배치 파일이 아닙니다.


02. 환경설정
    - 탐색기   - path - 연결

    - C:\Java1.8\jdk1.8\bin
    - JAVA_HOME: %시스템 변수%    // %% 시스템 변수를 나타내는 기호
    - JAVA_HOME 변수를 만들고 해당 변수에 경로값을 입력  C:\Java1.8\jdk1.8\bin 
    - 시스템경로에 들어가서 %변수명% 형태로 경로를 지정
